¿Qué es la red Lightning Network?

2022-05-24

Lightning Network que propone una nueva red, una segunda capa.

  • Se pueden realizar pagos entre usuarios de igual a igual.
  • No se necesita que se publicar una transacción en la cadena de bloques de Bitcoin para cada pago.
  • Los usuarios pueden pagarse entre sí en Lightning Network tantas veces como quieran sin incurrir en tarifas en cadena.
